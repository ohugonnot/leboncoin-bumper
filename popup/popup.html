Comment marche le tri ?
  • 🚨 Scam — patterns d'arnaque détectés (mandat cash, QR, numéro étranger, WhatsApp, lien externe…)
  • 💬 Lead — message substantiel, vraie question → à traiter
  • ❓ Question — "toujours dispo ?" et variantes, faible valeur
  • 🗑 Spam — message vide ou émoji seul

Archiver = cacher de la liste, local uniquement. Aucun impact côté Leboncoin. Visible dans l'onglet 📦 Archivés.

Le tri est heuristique : un faux positif scam est possible si le message contient un mot-clé sensible. Survole le détail "Détecté : …" sous chaque card pour comprendre.

Comment marche le score ★ ?
  • Chaque mot-clé matché dans le titre → +2 × poids
  • Matché seulement dans la description → +1 × poids
  • Le titre contient cherche / recherche / besoin / aide / demande / quelqu'un / conseil → +1
Poids par défaut : 1. Pour booster un mot-clé important, écris recalbox:3 (poids 3).
Survole le ★ d'une carte pour voir le détail du calcul.
Score min = filtre d'affichage. Notif si score ≥ = seuil pour t'alerter.
