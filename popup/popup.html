Comment marche le score ★ ?
  • Chaque mot-clé matché dans le titre → +2 × poids
  • Matché seulement dans la description → +1 × poids
  • Le titre contient cherche / recherche / besoin / aide / demande / quelqu'un / conseil → +1
Poids par défaut : 1. Pour booster un mot-clé important, écris recalbox:3 (poids 3).
Survole le ★ d'une carte pour voir le détail du calcul.
